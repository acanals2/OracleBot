Operational kit for landing **3–5 paid engagements** with a **$25k Standard** default for fintech / trading ICPs.

## First-quarter targets (from outbound doc)

- ~50 prospects sourced, 50 first touches, ~30 follow-ups.
- Expect ~8–12 replies, 4–6 calls, **1–2 closed engagements**.
- **Lead offer:** Standard **$25k** for trading platforms 30–90 days from launch; Pilot **$10k** when they need API-only proof first.
