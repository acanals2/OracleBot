# First-touch playbook

## Cadence (per prospect)

| Day | Action |
|-----|--------|
| 0 | Template **A** (email) or **B** (LinkedIn) from [docs/03-outbound-playbook.md](../docs/03-outbound-playbook.md) |
| 5 | Template **D** follow-up (three concrete failure modes tailored to them) |
| 15 | Template **E** (“last note,” permission to stop) |
| 30 | Nurture only (quarterly 1:1 “saw this and thought of you”) |

## Discipline

- **State price** in first touch ($25k Standard; $10k Pilot for crypto-native template C).  
- **Never** lead with “AI personas.” Differentiation is **population-scale behavior** on **verified staging**.  
- **Never** promise human-like behavior on every page of any site (V1 scope).  
- **Stagger** sends: don’t blast &gt;5 similar emails in one hour.  
- **Customize** line 1 with a real signal (funding, job post, testnet, SRE hire).

## Collateral order on calls

1. Problem: empty staging ≠ tested staging; seed scripts ≠ population interaction.  
2. Guardrails: DNS/OAuth gate; production refusal; audit trail — [Commitments on the site](../index.html#commitments).  
3. Deliverable: written readiness report; show [sample-readiness-report.html](../sample-readiness-report.html).  
4. Close: Standard **$25k** scope from [docs/02-concierge-offering.md](../docs/02-concierge-offering.md) or Pilot if they need a smaller first step.

## First $25k Standard “win path”

- Prospect: pre-launch **trading or signals** platform, named technical buyer.  
- Outcome: signed SOW referencing Standard tier, staging URL to verify, kickoff within **4–6 weeks**.  
- Track in **pipeline-tracker.csv** with `Closed_WON_LOST=WON` and `Amount=25000`.
